MTCG Rewards
The basics on how to use the reward game system.

Login
To earn rewards you need to "login" to the game miner. This records the wallet ID for the earned rewards.
Copy your wallet reciver address and paste it in the wallet id area of the login form.
You can leave the player name and reward token blank if you wish.
We recommend using a MintMe or MetaMask Wallet to claim blockchain rewards.

Earning Rewards
Once you have logged into the miner a timer will start calculating your rewards.
Feel free to play some of the miner games during this time.
Each game mines MintMe for the token associated for the game if they have a miner set-up.

Claiming the Token
Once you have earned one token in time, you can cashout in the menu and claim your reward.

Timer
The timer is set to 0.01 tokens per 3.141 seconds. It also has a pool booster the resets at 100 tokens.

Logs
Rewards are logged on the miner for display and sent on a monthly base, or if the log is full.
Cashout maybe disabled if the rewards pool is maxed out.
Once you have claimed your rewards you will be logged out and sent back to the home screen.
Remember if you use MetaMask you will need to pay a gas fee to use the token after it is sent to you.


Token Contracts
